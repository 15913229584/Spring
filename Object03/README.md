#创建一个代理对象，
但代理对象需要传入一个接口，而接口有需要set方法注入
代理对象步骤---->1.创建一个任意dao类型的代理对象
               2.使用set方法注入任意dao的Class对象
               3.（进行数据库的查询操作）调用getObject方法通过返回的工厂类的SqlSession的getMapper将代理对象放进容器里面。
               4.返回代理对象的接口，代理对象执行方法
               5.是否单例设置为是
               
               容器中：
               自定义Bean里代理对象代替用户实现（set注入）用户查询的方法
